# Ethereum USDT Wallet

A Python tool for generating Ethereum wallets and managing USDT (ERC-20) transactions.

## Features
- Generate Ethereum wallet (address + private key)
- Check ETH and USDT balance
- Send USDT via Web3

## Requirements
```
pip install web3
```

## Usage
```bash
python wallet.py --generate
python wallet.py --balance <address>
```
